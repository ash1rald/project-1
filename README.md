# Викторина - Quiz Application

**Версия:** 1.0.0 [RELEAESE]

## Описание

Интерактивное веб-приложение для прохождения викторины с отслеживанием по количеству правильных ответов.

## Возможности

- ✅ Мультивопросная викторина (quiz1, quiz2 и далее)
- ✅ Отслеживание счёта (localStorage)
- ✅ Страницы результатов для правильных/неправильных ответов
- ✅ Финальный результат с подсчётом баллов
- ✅ Кнопка "Начать заново" для перезагрузки

## Структура проекта

```
project#1/
```

## Использование

1. Откройте `main.html` в браузере
2. Нажмите "Начать викторину"
3. Выберите правильный или неправильный ответ
4. Посмотрите результат и переходите к следующему вопросу
5. После всех вопросов — финальный результат с опцией "Начать заново"
